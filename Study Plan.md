# 计划
## 每日“到小时”的可执行模板（工作日）

### A. 普通日（最常见）

07:20–08:00 洗漱/早餐/路上听英语（雅思听力或跟读）

08:00–12:00 上课

12:00–13:30 午饭+休息（别学习，防崩）

13:30–14:10 轻学习：背雅思词/口语素材（40min）

14:10–17:00 校内事务/作业/课程（机动）

17:00–18:00 晚饭+走路（放松）

19:30–20:30 雅思主训练（听/读/写轮换）

20:40–22:10 AI/编程学习或项目（90min）

22:10–22:30 复盘+记录（写到 GitHub：学习笔记/日志）

23:00 前 睡（别熬过头，熬夜会断更）

### B. 课程爆炸日（很累/作业多）

最低保命版（总共 60 分钟）：

20:30–21:00 雅思 30min（精听或阅读一篇）

21:00–21:30 项目 30min（只做一件小事并 commit）

这套的目的：不断档。断档一周，很容易彻底滑坡。

## 每周节奏（固定“周计划模板”）

周一：雅思写作（Task 2）+ 改错

周二：ML 学习 + 小练习（sklearn）

周三：雅思听力精听 + 口语

周四：项目推进（只做项目）

周五：复盘周任务 + 写周总结（发 GitHub/博客）

周六：项目大推进（3–5小时）+ README 更新

周日：雅思模拟（2小时）+ 休息半天（必须）

## 18个月“阶段任务清单”
### 月1–2：起步必须完成

建 GitHub 主页仓库（portfolio）

Python 基础 + pandas/numpy

雅思开始：每天至少 40–60min

项目#1 开始（回归/分类任选）

### 月3–4：第一次见成果

项目#1 完整上线（可复现）

雅思第一次考试（建议第4个月就考一次摸底）

给老师发邮件争取加入任务（用上面的模板）

### 月5–6：进入“可申请体质”

项目#2 上线

雅思冲到 6.5（至少单项不崩）

如果老师给任务：每周汇报一次，开始积累推荐信关系

### 月7–9：深度学习上强度

PyTorch 项目#3 上线

开始接触 Transformer/HuggingFace

简历/PS 初稿开始写（中文先写）

### 月10–12：大模型工程作品成型

RAG 项目#4 上线（强烈建议）

项目写 1 篇“架构解释”文章（申请很加分）

雅思最好到 7.0（如果到不了，6.5也能申请很多）

### 月13–15：申请启动

学校清单最终版（8–12所）

SOP 定稿、CV 定稿、推荐信锁定

奖学金清单逐个申请（别偷懒）

### 月16–18：面试与结果期

面试准备：项目讲解稿背熟（3个项目每个3分钟）

继续保持更新：每两周至少 1 篇笔记/1 次项目改进
